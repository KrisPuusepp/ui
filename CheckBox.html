Class: CheckBox

CheckBox

Creates a container based checkbox element

Example

 ```
  new CheckBox({
     checked: false,
     style: {
         unchecked: Sprite.from(`switch_off.png`),
         checked: Sprite.from(`switch_on.png`),
     }
 });

 ```
